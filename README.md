# OBS YouTube Uploader (WIP)
OBSで録画した動画を自動でYouTubeにアップロードするデスクトップアプリケーション

## 機能
- OBSの録画を遠隔操作
- YouTubeへの自動アップロード

## 必要な物
- Python 3.12
- OBS Studio
- YouTubeチャンネル
- Google Cloud プロジェクト（YouTube Data API v3）
